Ballot for TIGER-XML working group

Choices

For each proposal, you must select the course of action that you think is most appropriate by putting an "x" in the appropriate column:
  • Turn into recommendation: the NTN should turn the proposal into a recommendation as it is.
  • Set down working group: the NTN should set down a working group that will make a new recommendation based on this proposal (possibly combining it with other proposals as well); the new recommendation will be submitted to a second round of voting.
  • No action: the NTN cannot realistically make a recommendation that will suit a large number of NTN members, within the limited timespan of the network.
Within each area, you can only select the option "Turn into recommendation" for at most one proposal.
